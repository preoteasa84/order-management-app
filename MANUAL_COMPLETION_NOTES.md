## Usage

### Viewing in Browser
Open `MANUAL_UTILIZARE_COMPLET.html` in any modern browser (Chrome, Firefox, Edge)

### Printing
1. Open in browser
2. Press Ctrl+P (or Cmd+P on Mac)
3. Select printer or "Save as PDF"
4. Ensure orientation is set to **Landscape**
5. Ensure "Background graphics" is enabled for colors
6. Print or save

### Customization
